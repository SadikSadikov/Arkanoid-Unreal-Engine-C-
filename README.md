🧱 Arkanoid Clone (Unreal Engine C++)

A personal project built in Unreal Engine, focused on recreating a classic Arkanoid-style game while practicing C++ gameplay programming.

🎯 Project Overview

This project is primarily developed for learning and experimentation purposes.
The goal is to explore Unreal Engine systems, improve C++ architecture, and implement gameplay mechanics from scratch.

⚠️ This is not intended for commercial release (e.g. Steam), but rather as a learning resource.

⚙️ Features
💥 Brick destruction system
🎁 Power-up system (data-driven)
💾 Save system (High Score)
🤖 Basic Enemy logic (not yet integrated into gameplay)
🛠️ Customization

Most gameplay values can be easily tweaked via the editor:

Paddle speed
Ball speed
Power-up behavior
Brick layouts
📦 Data-Driven Design
PowerUps are configured using Data Assets
Brick patterns are also defined via Data Assets for flexible level design
⚠️ Known Issues
Ball movement uses custom bounce logic
→ currently has some bugs (to be improved)
